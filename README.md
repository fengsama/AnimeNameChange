# 影视文件批量重命名工具

一款简约轻量级的影视文件批量重命名工具，专为优化NAS存储环境下Emby媒体服务器的识别与刮削效果而设计。

## 功能特点

- **批量重命名**：支持处理多种常见视频格式文件
- **直观GUI**：友好的图形用户界面，支持拖拽添加文件
- **命名规则**：提供常用命名模板及自定义规则
- **实时预览**：在实际执行前展示重命名效果
- **错误处理**：包含文件权限检查、重名文件处理策略
- **操作日志**：记录重命名历史和结果
- **零环境依赖**：单文件可执行程序，直接双击运行

## 支持的视频格式

- MP4
- MKV
- AVI
- MOV
- WMV
- FLV
- TS

## 使用方法

1. **添加文件**：点击"添加文件"按钮选择单个文件，或点击"添加文件夹"按钮选择包含视频文件的文件夹
2. **拖拽添加**：直接将文件或文件夹拖拽到文件列表区域
3. **配置命名规则**：
   - 从下拉菜单选择预设模板
   - 或在自定义规则输入框中输入自定义格式
   - 填写影视类型、标题、季数、集数等信息
4. **预览效果**：在预览区域查看重命名后的效果对比
5. **开始重命名**：点击"开始处理"按钮执行重命名操作
6. **查看日志**：点击"查看日志"按钮查看操作历史

## 命名规则模板

- `[影视类型] - [标题] - [季数]x[集数]`
- `[标题] - [季数]x[集数]`
- `[影视类型] - [标题]`
- `[标题]`

## 自定义命名规则

您可以在自定义规则输入框中输入任意格式，支持以下占位符：

- `[影视类型]`：影视类型（番剧、电视剧、电影、其他）
- `[标题]`：影视标题
- `[季数]`：季数
- `[集数]`：集数

例如：`[标题] - [年份]`

## 注意事项

- 请确保对目标文件有写入权限
- 重命名操作会直接修改文件名，请谨慎操作
- 对于重名文件，系统会自动添加时间戳后缀

## 系统要求

- Windows 10 及以上操作系统
- 无需安装Python或其他依赖

## 常见问题

### Q: 为什么某些文件无法重命名？
A: 可能的原因包括：文件被其他程序占用、没有文件权限、目标路径不存在等。请查看操作日志获取详细错误信息。

### Q: 如何保存我的命名规则配置？
A: 点击"保存配置"按钮，系统会将当前配置保存到config.json文件中，下次启动时会自动加载。

### Q: 重命名后的文件格式不符合我的预期？
A: 请检查命名规则配置，确保占位符使用正确，并且填写了必要的信息（如标题）。
